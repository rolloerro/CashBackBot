# 💸 CashBackBot

Telegram-бот **CashBackBot** — это минималистичный каркас сервиса для агрегации купонов, акций и кешбэков с онлайн-площадок и источников.

Проект создан как **MVP / Skeleton**, который легко расширяется и дорабатывается под конкретные задачи.

---

## 🚀 Что делает бот сейчас

- Работает в Telegram
- Показывает список акций / купонов из локального источника (JSON)
- Использует inline-кнопки
- Простая и прозрачная логика
- Не требует регистрации и сложной настройки
- Работает 24 часа в сутки
- На всех устройствах ( кроссплатформенность) 

---

## 🧱 Архитектура (осознанно простая)

- **Python**
- **aiogram**
- **JSON / SQLite** (по желанию)
- **asyncio**
- Поддержка планировщиков (cron / background tasks)

Проект не перегружен логикой специально — это **чистый старт**, а не «комбайн». 

---

## 🔌 Что можно подключить дальше

Вы можете легко добавить:
- RSS-ленты магазинов и агрегаторов
- Парсинг сайтов и маркетплейсов
- Персональные подборки по категориям
- AI-рекомендации и фильтрацию
- Уведомления по расписанию
- Платные подписки / партнёрские программы

Бот спроектирован так, чтобы **всё это подключалось без переписывания ядра**.

---

## ⚠️ Важно

Это **каркас и тестовый бот**.  
По умолчанию внешние источники (RSS, API магазинов) **не подключены**.

Если вы хотите:
- подключить конкретные магазины
- настроить RSS
- доработать под свой бизнес-кейс  

— всё это делается индивидуально.
